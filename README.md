# AI vs Human 文章分類工具 🤖

一個使用 Streamlit 建構的 AI 內容偵測工具，能夠分析文本並判斷其是否由 AI 生成。

## 功能特色

### 1. 即時文本分析
- 使用者輸入文本後立即顯示 AI% / Human% 判斷結果
- 支援中文和英文文本分析
- 提供置信度評估（高/中/低）

### 2. 多種分析方法
- **整合分析 (Ensemble)**: 結合多種方法的綜合判斷（推薦）
- **特徵分析法 (Feature-based)**: 基於文本特徵的分析
- **統計分析法 (Statistical)**: 基於統計分布的分析

### 3. 自建特徵提取
提取 15+ 種文本特徵：
- 詞彙豐富度 (Vocabulary Richness)
- 句子長度變異性 (Sentence Length Variance)
- 過渡詞使用率 (Transition Word Ratio)
- 突發性指標 (Burstiness)
- 標點符號比例
- N-gram 重複模式
- Zipf 定律偏離度
- 等等...

### 4. 可視化與統計量
- **儀表板圖表**: 直觀顯示 AI/Human 機率
- **雷達圖**: 展示各項特徵的分析結果
- **詞長分布圖**: 文本詞彙長度分布
- **統計資訊表**: 完整的文本統計數據

pip install -r requirements.txt
```

streamlit run app.py
```

應用程式將在瀏覽器中自動開啟，預設網址為 `http://localhost:8501`

## 技術架構

### 使用的函式庫
- **Streamlit**: Web 應用框架
- **NumPy & Pandas**: 資料處理
- **Plotly**: 互動式圖表
- **scikit-learn**: 機器學習工具

### AI 偵測原理

本工具基於以下觀察來判斷文本是否由 AI 生成：

1. **詞彙多樣性**: AI 生成的文本通常有較高的詞彙多樣性
2. **句子一致性**: AI 文本的句子長度變化較小
3. **過渡詞使用**: AI 傾向使用較多的過渡詞（however, therefore 等）
4. **突發性 (Burstiness)**: 人類寫作中詞彙出現更具突發性
5. **Zipf 定律**: AI 文本的詞頻分布更接近理想 Zipf 分布

## 使用說明

1. 在文本框中貼上要分析的文章（建議至少 100 字）
2. 選擇分析方法（預設為整合分析）
3. 點擊「開始分析」按鈕
4. 查看分析結果與詳細特徵

## 注意事項

⚠️ **免責聲明**:
- 此工具僅供參考，結果不能作為絕對判斷依據
- 偵測準確度受限於文本長度和內容類型
- 建議與其他偵測工具交叉驗證
